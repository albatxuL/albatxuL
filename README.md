## Sobre mí

Desarrollo herramientas para videojuegos con Unity/C# y aplicaciones web full-stack con .NET y Angular.  
Me especializo en **tools programming**, arquitecturas **data-driven** y sistemas **editor-friendly**.  
Me interesa especialmente la **IA aplicada a videojuegos**: comportamiento de agentes, sistemas procedurales y toma de decisiones.
